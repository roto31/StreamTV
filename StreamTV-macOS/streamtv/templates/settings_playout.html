Playout Settings

Configure how many days ahead playout schedules are built for EPG and streaming.

Schedule Build Configuration
Number of days ahead to build playout schedules. This affects EPG (Electronic Program Guide) generation and how far in advance programming is planned. Range: 1-14 days. Default: 1 day (24 hours).
info Information

Playout Days To Build: This setting controls how many days into the future StreamTV generates playout schedules. This directly affects:

  • EPG Generation: The XMLTV guide (used by Plex, Emby, Jellyfin) will show programming for the specified number of days ahead.
  • Schedule Planning: Channels will have their programming planned and available for streaming for the specified number of days.
  • Performance: Higher values (more days) will increase EPG generation time and memory usage, but provide more programming information to clients.

Recommended Values:

  • 1 day: Good for testing and low-resource systems. EPG updates quickly.
  • 3-7 days: Recommended for most users. Provides a good balance of information and performance.
  • 14 days: Maximum recommended. Useful for users who want extensive programming information, but may take longer to generate.

Note: Changes to this setting take effect immediately on the next EPG request. No server restart required.
